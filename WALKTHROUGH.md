## 2. 如何开启你的写作之旅？

### 第一步：安装技能 (必须)
请在终端执行以下命令来安装你的专属技能：
```bash
gemini skills install suanlirongyu-writing.skill --scope user
```
**重要**：安装后，请在交互界面输入 `/skills reload` 激活它。

### 第二步：典型互动场景

#### 场景 A：看到一篇硬核好文，想写拆解
你：`分析素材 https://example.com/deepseek-tech-report`
Agent：(输出洞察卡片，提炼核心矛盾)

#### 场景 B：有了几个碎片的灵感，想出大纲
你：`构思大纲：1. 算力成本正在下降 2. 边缘侧 AI 爆发 3. 隐私计算成为刚需`
Agent：(输出结构化大纲，建立逻辑脉络)

#### 场景 C：大纲已定，一键成稿
你：`确认大纲。请撰写全文，注意保持「算力冗余」那种冷峻的语气，不需要结尾总结。`
