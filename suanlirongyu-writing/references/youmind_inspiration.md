# YouMind 灵感：优秀写作提示词与技能参考

在调研 YouMind 及其背后的创作理念后，我们提炼了以下可以增强 Agent 能力的“写作秘籍”：

## 1. “万物化稿”的素材处理 (Input)
YouMind 的核心在于将各种媒介转化为创作养分。
- **视频/播客转稿 Prompt**: "忽略所有的口语词和语气词，提取这段音频中的 3 个核心论断，并按时间轴标注其出现的逻辑转折点。"
- **PDF 论文拆解 Prompt**: "不要只总结摘要。请找出作者在方法论部分提出的 3 个关键假设，并分析如果这些假设失效，结论会受到什么影响。"

## 2. “第二大脑”的关联思考 (Process)
YouMind 强调资料间的“互联”。
- **跨领域关联 Prompt**: "基于我今天输入的关于【大模型效率】的素材，尝试将其与我上周保存的【能源危机】素材进行关联，推演算力增长对全球电力市场的潜在冲击。"
- **反向论证 Prompt**: "目前的大纲看起来逻辑顺滑，请扮演一个最苛刻的读者，找出大纲中 2 处‘想当然’的逻辑跳跃，并提出补证方案。"

## 3. “沉浸式”写作风格控制 (Output)
- **视觉化描述 Prompt**: "在描述这个技术架构时，请使用‘像……一样的……’这类具体比喻，但必须限制在工业/工程领域，严禁使用自然/生活类比喻。"
- **节奏控制 Prompt**: "确保段落长度呈现 1-3-1 的节奏。即：一句有力的开场词，三句深度解析，一句引发思考的短句结尾。"

## 4. 快捷指令设计 (Shortcuts)
参考 YouMind 的 Shortcuts，你可以直接对我喊：
- `/digest`: 帮我总结当前链接，出洞察卡片。
- `/mindmap`: 帮我把现在的碎片灵感画个逻辑图（用 Markdown 列表表示）。
- `/polishing`: 用「算力冗余」最冷峻的语气重写这一段。
