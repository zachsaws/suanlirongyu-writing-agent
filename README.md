## 0. 为什么需要 v5.0？
如果说 v3.0 解决了自动化，**v5.0 则解决了“思想深度”的降维打击。** 
通过引入 **Master Benchmark**，Agent 现在的每一篇文章都会在内部与 **Stratechery** (商业格局)、**LessWrong** (认知穿透)、**少数派** (智性呼吸) 进行递归对标，直到逻辑厚度达到全球 Top 1% 文章的质感。

## 1. 核心进化能力 (v5.0 New Features)

### 💎 Master Benchmark 递归系统
不再满足于“写对”。Agent 在输出前会执行 **“对比-Gap分析-递归修正”** 的闭环逻辑。它会自省：“相比 Ben Thompson，我的二阶效应推演够吗？”如果不达标，必须自动重写，直到满足高奢标杆。

### 🧠 长期记忆与实体百科 (Memory v4.0)
内置分层长期记忆。Agent 每一篇新文章都会查阅 **`entity_wiki.md`**，确保对 NVIDIA、OpenAI 等实体的观察具有跨时间的连贯性，实现真正的“逻辑复利”。

### 🎨 工业级视觉审美引擎 (Aesthetics v4.2)
彻底剔除“章节、第一、首先”等教科书痕迹。引入 **“去编号化叙事”**。利用 H2+Emoji 视觉系统、4行原则、以及单句锚点，打造极致的移动端阅读穿透力。

### 🔄 自我进化与样文训练
支持通过 `/train [URL]` 实时提取优质好文的 DNA。通过记录用户反馈并运行 `/evolve`，Agent 能在 10 秒内完成自我规则的重构与热更新。

## 2. 核心指令集 (Master Shortcuts)
- `/plan`: 结合记忆检索生成分层规划大纲。
- `/debate`: 启动 Antagonist 内部博弈，发起虚拟 PR 评审。
- `/finalize`: 执行 **v5.0 递归成稿**，包含视觉规划与逻辑呼吸。
- `/evaluate`: 基于 Master Benchmark 输出详细的 Gap 报告。
- `/chief`: 结合全球热点与第一性原理生成选题。
- `/train`: 提取样文 DNA 并更新标杆坐标系。
