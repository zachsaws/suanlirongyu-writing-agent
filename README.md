## 0. 核心定位
**v7.1** 是在经历了复杂系统熵增后的“去油”巅峰之作。它放弃了繁琐的指标堆砌，将力量集中在**「穿透力」**和**「非共识」**上。它不仅是一个助手，更是一个具备独立审美和长期记忆的“数字主编”。

## 1. 核心灵魂能力 (Key Features)

### 💎 共识回避机制 (Non-consensus Mechanism)
Agent 被强制要求先识别全网平庸共识，并执行“90度夹角避让”，确保每一篇输出都具备全网唯一的独家切口。

### 🧠 长期记忆与实体百科 (Memory v4.0)
内置 `memory/` 空间，持续累积对 NVIDIA、OpenAI 等实体的历史定论，确保观点具有跨时间的连贯性与逻辑复利。

### 🎨 极简工业叙事 (Narrative v4.2)
彻底告别“第一章、首先、总之”等陈旧痕迹。采用 **4行段落原则**、**单句锚点加粗** 以及 **无编号流动叙事**，适配顶级移动端阅读审美。

### 🔄 实证驱动的 Benchmark
对标 **Stratechery** (格局)、**LessWrong** (深度)、**少数派** (呼吸感)。成稿自带 **`<Self-Evolution Report>`**，透明展示逻辑熵与非共识点的达成度。

## 2. 核心指令集
- `/plan`: 结合历史记忆与共识回避逻辑，生成硬核大纲。
- `/finalize`: 执行 v7.1 最终成稿，包含视觉规整与自省报告。
- `/chief`: 结合第一性原理扫描全球热点选题。
- `/evolve`: 提取用户近期反馈，瞬间完成技能包的热更新。
